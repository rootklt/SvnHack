### SvnHack ###

SVN信息泄漏利用脚本，适用于svn version 1.7。

---

### Usage ###

---

```bash 
python3 SvnHack.py -h
```
![var](https://github.com/rootklt/SvnHack/help.gif)

```bash
python -u http://127.0.0.1:8080/.svn --download

#下载.svn目录下的源代码
```
